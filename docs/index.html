Quick Start

Installation

1

Clone & Build

cd wormhole
npm install
npm run build
2

Configure Claude Desktop

Add to ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "wormhole": {
      "command": "node",
      "args": ["/path/to/wormhole/dist/server.js"]
    }
  }
}
3

Ready!

Restart Claude Desktop and start using Wormhole.
